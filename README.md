## ᚱ Credits & Thesis Summary

### Total Credits by Level
- **BS (x13)**: 128–132 credits each → ~1,680 total BS credits
- **MS (x4)**: 30–36 credits each → ~132 total MS credits
- **PhD (x5)**: 72–96 credits each (including dissertation) → ~420 total PhD credits
- **Grand Total**: ~2,232 credits

### Thesis/Dissertation Requirements
- **BS**: Capstone project or thesis (8–16 credits)
- **MS**: Thesis or capstone project (6 credits)
- **PhD**: Original research dissertation (60,000–120,000 words) + oral defense
