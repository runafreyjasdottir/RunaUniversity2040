## ᚱ Core Theories & Frameworks

### The OSI Model (Still Relevant in 2040)
Despite quantum and 6G disruption, the 7-layer model remains the conceptual framework:
1. **Physical** — fiber, wireless, quantum channels, satellite links
2. **Data Link** — Ethernet, WiFi 7, 6G radio, LoRa
3. **Network** — IPv6, segment routing, BGP, quantum routing
4. **Transport** — QUIC (replaces TCP for most), SCTP, multipath
5. **Session** — TLS 1.4, quantum key distribution
6. **Presentation** — protobuf, CBOR, schema registries
7. **Application** — HTTP/3, gRPC, MQTT, CoAP

### Key Protocols (2040 Landscape)
- **QUIC** — the default transport (replaces TCP+TLS)
- **HTTP/3** — QUIC-based, mandatory encryption
- **gRPC** — high-performance RPC, protobuf-encoded
- **BGP++** — enhanced with RPKI, route leak detection, quantum-secure
- **Segment Routing (SRv6)** — source-routed, programmable forwarding
- **WireGuard** — minimal VPN protocol, PQC-upgraded
- **Zigbee/Matter** — IoT mesh protocols
- **MQTT 6.0** — AI-native pub/sub for edge

### Network Design Principles
- **Defense in depth** — multiple security layers, zero-trust at every hop
- **Modularity** — clear failure domains, blast radius containment
- **Scalability** — horizontal scaling, Clos topologies, leaf-spine
- **Observability-first** — telemetry from day one, not bolted on
- **Automation imperative** — network as code, GitOps for configs

## ᛏ 2040-Era Advances

### Quantum Networking
- **Quantum Key Distribution (QKD)** — entanglement-based, impossible to intercept without detection
- **Quantum repeatless** — no amplification needed, quantum memory relays
- **Hybrid classical-quantum links** — classical channel + quantum channel side-by-side
- **Quantum internet** — early research networks connecting quantum processors across continents
- **Post-quantum TLS** — CRYSTALS-Kyber for key exchange, CRYSTALS-Dilithium for signatures

### 6G & Beyond
- **Terahertz communication** — 100 Gbps+ on short-range links
- **Intelligent surfaces** — reconfigurable metasurfaces that shape RF propagation
- **Integrated sensing & communication** — network doubles as radar/lidar
- **AI-native air interface** — ML-designed waveforms, not human-specified
- **Satellite mega-constellation mesh** — low-orbit satellites forming a global backbone

### Software-Defined Everything
- **SD-WAN 3.0** — AI-driven path selection, predictive bandwidth allocation
- **SASE (Secure Access Service Edge)** — converged networking + security
- **Intent-based networking** — describe what you want, AI configures it
- **Digital twins** — simulate network changes before deploying
- **Self-optimizing networks** — continuous route optimization via reinforcement learning
