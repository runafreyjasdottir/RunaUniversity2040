# Bachelor of Science in Computer Science — University of Yggdrasil, 2040

## ᚠ Overview: The Foundations of Computation

The Computer Science degree at UoY has evolved dramatically from its 2020s roots. By 2040, the discipline spans classical computation, quantum computing, neuromorphic architectures, and AGI-adjacent theory. The core philosophy remains: **understand computation at its deepest level so you can shape it, not just use it**.

Graduates emerge with the ability to reason about computation itself — from Turing machines to quantum circuits to self-modifying neural architectures — and to build systems that transcend current paradigms.

## ᚱ Core Theories & Frameworks

### Computational Complexity
- **P vs NP** — still open in 2040, but practical quantum advantage reshapes the landscape
- **Complexity Zoo** — P, NP, co-NP, PSPACE, EXP, BQP, QMA — the full hierarchy
- **Parameterized complexity** — fixed-parameter tractability for real-world problem sizes
- **Average-case analysis** — smoothed analysis, beyond worst-case

### Type Theory & Formal Verification
- **Martin-Löf type theory** — the foundation of verified computing
- **Homotopy Type Theory (HoTT)** — univalent foundations for mathematics and programs
- **Linear types** — Rust's ownership model generalized (affine, relevant, linear)
- **Dependent types** — Agda, Idris 2, Lean 4 for proof-carrying code
- **Effect systems** — tracking side effects algebraically (Koka, Eff)

### Information & Coding Theory
- **Shannon entropy** — the bedrock of all compression and communication
- **Error-correcting codes** — Reed-Solomon, LDPC, polar codes, quantum error correction
- **Rate-distortion theory** — optimal lossy compression bounds
- **Network coding** — beyond store-and-forward routing

### Learning Theory
- **PAC learning** — probably approximately correct framework
- **VC dimension** — capacity of hypothesis classes
- **Rademacher complexity** — modern generalization bounds
- **Neural tangent kernel** — infinite-width limit theory for deep networks
- **Scaling laws** — Chinchilla-optimal compute allocation, emergent capabilities

## ᛏ 2040-Era Advances

### Quantum-Enhanced Computing
- **QPU coprocessors** — quantum processing units as accelerators alongside classical GPUs
- **Variational Quantum Eigensolver (VQE)** — chemistry and materials simulation
- **Quantum Machine Learning** — hybrid quantum-classical training pipelines
- **Post-Quantum Cryptography (PQC)** — lattice-based (CRYSTALS-Kyber/Dilithium) now standard
- **Quantum networking** — entanglement-based key distribution in production

### AI-Native Development Paradigms
- **Prompt-first architecture** — systems designed around LLM orchestration layers
- **Agent frameworks** — MCP (Model Context Protocol), ACP, autonomous tool use
- **Synthetic data pipelines** — generating training data from foundation models
- **Neural program synthesis** — specification→implementation via LLM+verification
- **Self-healing code** — runtime error detection and autonomous patch generation

### Post-REST Architectural Patterns
- **GraphQL Federation** — unified schema across microservices
- **gRPC-streaming services** — bidirectional streaming, protobuf efficiency
- **Event-driven mesh** — async message passing, exactly-once semantics
- **Edge-cloud continuum** — compute at data source, sync to cloud when needed
- **Zero-knowledge service mesh** — provable privacy between services

### AGI-Safe Code Practices
- **Formal specification** — contracts, invariants, and pre/post-conditions verified at compile time
- **Sandboxed agent execution** — WASM/EBPF isolation for AI-generated code
- **Capability-based security** — every module declares minimum privileges
- **Interpretability-first design** — causal tracing built into every pipeline
- **Corrigibility protocols** — graceful shutdown, uncertainty signaling, override chains

## ᛞ Capstone Requirements

Students must complete a two-semester capstone:
1. **CS406**: Architecture document, design review, prototype demo
2. **CS407**: Full implementation, test suite (≥90% coverage), deployment, public writeup

**Approved Capstone Themes (2040)**:
- Quantum-classical hybrid optimizer for NP-hard scheduling
- Verified compiler pass using Lean 4
- Multi-agent system with provable safety guarantees
- Neuromorphic edge inference pipeline for realtime sensor data
- Zero-knowledge proof system for privacy-preserving analytics
- Self-healing distributed database with AI-assisted recovery
