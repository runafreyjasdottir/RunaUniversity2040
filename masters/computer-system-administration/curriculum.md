## ᚱ Core Knowledge Domains

### Fleet-Scale Orchestration
- Multi-cluster Kubernetes Federation (KubeFed)
- Service mesh: Istio, Cilium, Linkerd
- GitOps: ArgoCD, Flux, progressive delivery
- Observability: OpenTelemetry, Prometheus, Grafana, Tempo
- Policy enforcement: OPA/Gatekeeper, Kyverno

### Zero-Trust Architecture
- Identity-based perimeter (no network trust)
- Service-to-service mTLS with PQC-ready certificates
- Microsegmentation via Cilium/eBPF
- Continuous posture assessment
- Just-in-time access (break-glass) for emergencies

### AIOps
- Anomaly detection on metrics/logs/traces
- Predictive capacity planning
- Automated incident response playbooks
- Self-healing reconciliation loops
- Cost optimization via usage analysis

### Quantum-Safe Infrastructure
- PQC migration planning (SPIFFE/SPIRE for workload identity)
- Hybrid TLS (classical + post-quantum)
- Crypto agility (algorithm negotiation without code changes)
- HSM lifecycle management
- Certificate automation (cert-manager, Let's Encrypt)

### Disaster Recovery
- Active-active multi-region
- RPO/RTO modeling
- Chaos engineering (Gremlin, Litmus)
- Backup strategy (3-2-1-1-0 rule)
- Runbook automation

## ᛞ Capstone Requirements
- Design a production-grade multi-region infrastructure
- Implement zero-trust networking, PQC migration plan, chaos engineering
- Operate for 8 weeks with monitoring and incident response
- Write comprehensive runbooks, postmortems, and architecture decision records
